## 🔧 技术实现

### 搜索模块

**文件**: `core/search.py`

**功能**:
1. **DuckDuckGo 搜索** - 免费，无需 API Key
2. **新闻搜索** - 专门搜索新闻
3. **网页内容抓取** - 提取网页文本

**特点**:
- ✅ 无需注册 API
- ✅ 完全免费
- ✅ 隐私保护（不追踪）
- ✅ 全球覆盖

### 自动搜索触发

**检测关键词**:
```python
search_keywords = [
    '新闻', 'news', '最新', 'latest', 
    '今天', 'today', '当前', 'current',
    '版本', 'version', '实时', 'real-time',
    '搜索', 'search', '天气', 'weather',
    '股票', 'stock', '价格', 'price',
    '汇率', 'rate'
]
```

当消息包含这些关键词时，**自动触发联网搜索**！

---

## 📊 工作流程

```
用户提问
   ↓
检测关键词
   ↓
触发搜索 (DuckDuckGo)
   ↓
获取实时结果
   ↓
作为上下文发送给 LLM
   ↓
LLM 生成回答（包含实时信息）
   ↓
返回给用户
```

## 🔍 API 端点

### 新增搜索 API

```python
# 网页搜索
GET /api/search/web?q=关键词

# 新闻搜索
GET /api/search/news?q=关键词

# 网页内容抓取
GET /api/search/fetch?url=网址
```

### Python 调用示例

```python
from core.search import search_web, search_news

# 网页搜索
results = search_web("AI 最新消息", num_results=5)

# 新闻搜索
news = search_news("今日科技新闻", num_results=5)
```

## ⚠️ 注意事项

### 搜索限制

1. **速率限制** - DuckDuckGo 有请求频率限制
2. **结果数量** - 每次最多返回 10 条结果
3. **语言支持** - 主要支持英文，中文搜索效果稍差

### 最佳实践

1. **具体问题** - 问题越具体，结果越准确
2. **关键词优化** - 使用关键词触发搜索
3. **结果验证** - 重要信息建议核实多个来源

### 隐私保护

- ✅ 不记录搜索历史
- ✅ 不追踪用户行为
- ✅ 不收集个人信息
